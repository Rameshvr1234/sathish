üöÄ Quick Start (After Download)

  1. Extract the ZIP file to your desired location
  2. Open terminal and navigate to the project folder
  3. Install backend: cd backend && npm install
  4. Install frontend: cd frontend && npm install
  5. Setup database: Create PostgreSQL database
  6. Configure .env files: Copy from .env.example
  7. Start backend: npm run dev
  8. Start frontend: npm start
